# 股票分析作品集（单股票）

本项目按四阶段落地一个“最小可运行股票数据产品”，默认示例为港股 `商汤-W (0020.HK)`。

## 第一阶段：MVP（跑通数据流程）

- 只做单一股票，不混合新闻等其他数据。
- 数据源：`Investing.com`（通过 `investpy` 获取）。
- 爬取字段：日 K 线（`Date`, `Open`, `High`, `Low`, `Close`, `Volume`）。
- 最小目标：获取最近 30 天历史价格数据。

## 第二阶段：数据清洗与分析（Pandas）

- 缺失值处理：前向填充 + 后向填充 + 兜底填 0。
- 统一格式：日期转标准日期格式，数值列转数值类型。
- 业务指标：
  - 日回报率：`Close_t / Close_{t-1} - 1`
  - 简单移动平均线：`SMA(5)`
  - 指数移动平均线：`EMA(12)`
  - 分析案例：过去 N 天中，涨幅大于 X% 的交易日占比

## 第三阶段：可视化与报告

- 主图：收盘价走势 + SMA + EMA
- 辅助图：成交额走势
- 资金估算：主力/小散成交额与净流入（分钟级估算口径）
- 报告输出：平均日回报率、最大单日涨幅、涨幅阈值占比
- 实时报价：最新价、涨跌幅、今开、今高、今低、昨收
- 美股扩展：盘前/盘后报价 + 实时K线（1m、3m、15m、30m、1h、1d）
- 美股状态灯：盘前 / 盘中 / 盘后 / 休市（按美东时间自动判断）
- 自动刷新：美股扩展页支持按秒级自动刷新

## 时事新闻模块

- 新闻来源：CNBC、Morgan Stanley、Investing.com（关键词可配置）
- 输出字段：标题、来源、发布时间、新闻链接
- 情绪标签：利好 / 中性 / 利空（基于关键词规则）
- 联动分析：可选择单条新闻并在价格图中标记事件点
- 影响评估：自动计算新闻后 1h / 4h / 1d 价格变化

## 第四阶段：部署与升级思路

- Web UI：使用 `Streamlit`
- 输入股票代码 -> 点击按钮 -> 展示图表与指标
- 定时任务说明：可接入 `Cron/Airflow`，每 6 小时自动执行
- 数据持久化：输出到本地 `CSV + SQLite`

## 项目结构

- `streamlit_app.py`：Web 界面入口（推荐，英为财情数据）
- `data_pipeline.py`：抓取、清洗、特征、分析、持久化
- `news_service.py`：新闻抓取与情绪标签
- `app.py`：此前实时看板 API（保留）
- `templates/index.html`：此前实时看板页面（保留）

## 安装与运行

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

运行后浏览器会自动打开本地地址（通常是 <http://localhost:8501>）。

> 说明：英为财情的网页数据接口可能存在频控或延迟，演示时建议间隔刷新。
